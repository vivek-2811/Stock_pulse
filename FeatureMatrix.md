# Product Feature Matrix

This matrix maps StockPulse pages to their core technical features.

---

## 1. Feature Map

| Feature Module | Business Value | Technical Highlights |
|---|---|---|
| **Dashboard** | Unified market overview | Live NY countdown, SVG sparklines |
| **Screener Pro** | Stock discovery engine | Opportunity scores, relative rankings |
| **Portfolio Hub** | Asset allocation | Zustand persist, tax ledger calculators |
| **AI Copilot** | Inline intelligence | Multi-module analysis generator |
| **News & Catalysts** | Bloomberg-like timelines | Scrollable 3-column workspace, breaking marquee |
| **Showcase Page** | Recruiter onboarding | Auto Demo navigation loop, data resetters |
| **Shortcuts Modal** | Accessibility cheatsheet | Shift + / global listener |

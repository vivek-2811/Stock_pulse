# StockPulse Pro Terminal — Institutional Market Intelligence Platform

StockPulse is a high-density, real-time simulated financial market intelligence workspace built to demonstrate senior software engineering maturity. The platform implements layouts, command centers, and AI analysis pipelines comparable to Bloomberg Terminal, TradingView, and Benzinga.

---

## 🚀 Recruiter Quick Start (Assess in Under 3 Minutes)

### 1. Auto Guided Demo Mode
1. Launch the application and click **Showcase** in the sidebar.
2. Click **Play Guided Demo**.
3. The platform will automatically drive itself through major screens (Dashboard → Portfolio → Screener → Intelligence → News → Copilot → Compare), overlays highlighting the tech stack and architectural decisions on each viewport.

### 2. Sandbox Reset Button
 recruiters will click random buy/sell buttons or change filters. Click **Reset Data baseline** on the Showcase page (or the header Reset button) to instantly restore all portfolio holdings, watchlists, saved filters, and AI chat logs to their pristine factory baselines.

### 3. Keyboard Shortcuts
* Press `?` (or `Shift + /`) at any time to open the **Keyboard Shortcuts Modal** cheat sheet.
* Press `Ctrl + K` (or `Cmd + K`) to open the **Raycast/Linear-like Command Center**, allowing you to execute navigation commands or search tickers.

---

## 🛠️ Technology Stack & Engineering Specs
* **Frontend**: React v19, React Router v7, Vite & Rolldown
* **State Management**: Zustand + Caching storage middlewares
* **Animations**: Framer Motion (spring mechanics, exit-entry layout morphs)
* **Styling**: Tailwind CSS v4 (modular variables, custom glassmorphism components)
* **Visualizations**: Canvas and SVG relative charts
